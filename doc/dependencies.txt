Dependencies
============

* Python_ ≥ 2.6

* argparse_ —
  Python command line parser

* Python Imaging Library (PIL_)

* Scanner Access Made Easy (SANE_) —
  API and tools to access to scanner hardware

* Jinja2_ -
  Python templating library 

.. _Python:
   https://www.python.org/
.. _argparse:
   https://code.google.com/p/argparse/
.. _PIL:
   http://www.pythonware.com/products/pil/
.. _SANE:
   http://www.sane-project.org/
.. _Jinja2:
   http://jinja.pocoo.org/docs/
